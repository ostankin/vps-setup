Ansible-плейбук для установки на VPS xray и squid в самом примитивном исполнении.

Требования к работе скрипта:

- Ansible 2.x
- Наличие доменного имени, для которого есть A-запись в DNS, указывающая на целевой сервер.
- наличие на целевом хосте рутового аккаунта с паролем (стандартная конфигурация VPS)

Плейбук разворачивает две системы:

1. Прокси-сервер с возможностью подключения к нему по HTTP (http://1.2.3.4:3128).
   Подключение без аутентификации, но доступ разрешён только с фиксированных IP-адресов.
1. Xray с доступом по личному ключу и верификацией через публичный ключ сервера.

Перед установкой в каталоге `group_vars` скопировать/переименовать `all.yml.example` в `all.yml` и заполнить все переменные актуальными значениями. Учитывая, что для генерации `xray_private_key` и `xray_public_key` потребуется уже установленный `xray`, эти значения можно временно оставить исходными,
но потом лучше сгенерировать их и перезапустить плейбук.

После установки нужно зайти на сервер и дать там команду `cat /etc/xray/clients.txt`
для получения VLESS-ссылок и QR-кодов для клиентов.

Тестировалось на мобильном клиенте
[v2rayNG версии 2.0.16](https://github.com/2dust/v2rayNG/releases/tag/2.0.16)
